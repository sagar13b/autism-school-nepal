# Assets Folder

This folder contains all images and media files for the website.

## Required Files

### Favicon Files
- `favicon.ico` - Main favicon (16x16, 32x32, 48x48)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 Apple touch icon

### Images
- `founder.jpg` - Photo of Dr. Arati Sharma (recommended: 400x400px, square)
- `og-image.jpg` - Open Graph image for social media sharing (1200x630px)
- `logo.png` - School logo (optional)

### Gallery Folder (`gallery/`)
Add photos of:
- Classrooms and learning spaces
- Students engaged in activities (with proper consent)
- Staff and therapists at work
- Facilities and equipment
- Events and celebrations

**Privacy Note:** Ensure you have proper consent from parents/guardians before using any photos of children. Consider using images that don't show faces or use stock photos of educational settings.

## Image Guidelines

- **Format**: JPG for photos, PNG for logos/icons
- **Optimization**: Compress images before uploading (use tools like TinyPNG)
- **Naming**: Use descriptive, lowercase names with hyphens (e.g., `classroom-activity-1.jpg`)
- **Alt Text**: All images should have descriptive alt text in the code

## Creating OG Images

For social media previews (og-image.jpg):
- Size: 1200x630px
